This is a tool for adding a password to any existing URL. In other words, for encrypting & decrypting links. Users will be prompted for entering the password whenever they visit an encrypted URL. If the password is correct, the application server redirects to the hidden website. Otherwise, an error is displayed. Users can also add hints to remind them of the password.
